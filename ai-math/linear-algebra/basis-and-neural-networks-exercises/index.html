    习题与思考：基变换与神经网络¶

    A 档巩固基础，B 档培养 ML 直觉。答案提示见各题末尾。

    四、基变换与神经网络（§12–§14）¶

    A 档¶

    A1. 设线性变换 \(T\) 在标准基下的矩阵为 \(A = \begin{bmatrix}1&2\\3&4\end{bmatrix}\)。求 \(T\) 在基 \(\mathcal{B} = \{(1,1)^{\mathsf T},\; (1,-1)^{\mathsf T}\}\) 下的矩阵表示 \(B\)。

    答案提示： 过渡矩阵（基向量按列排）\(P = \begin{bmatrix}1&1\\1&-1\end{bmatrix}\)，其逆 \(P^{-1} = \frac{1}{2}\begin{bmatrix}1&1\\1&-1\end{bmatrix}\)（\(P\) 自逆到缩放因子）。则：

    \[B = P^{-1}AP = \frac{1}{2}\begin{bmatrix}1&1\\1&-1\end{bmatrix} \begin{bmatrix}1&2\\3&4\end{bmatrix} \begin{bmatrix}1&1\\1&-1\end{bmatrix} = \begin{bmatrix}5&1\\-2&0\end{bmatrix}.\]

    这就是相似变换：\(B\) 与 \(A\) 表示同一个线性变换，只是从不同的「视角」（基）来看。它们有相同的特征值、迹和行列式。

    A2. 设 \(A\) 是一个 \(n \times n\) 矩阵，\(P\) 是可逆矩阵，\(B = P^{-1}AP\)。

    (1) 证明 \(\text{tr}(B) = \text{tr}(A)\)。（提示：\(\text{tr}(XY) = \text{tr}(YX)\)） (2) 证明 \(\det(B) = \det(A)\)。 (3) 若 \(A\mathbf{x} = \lambda\mathbf{x}\)（\(\mathbf{x} \neq \mathbf{0}\)），证明 \(\lambda\) 也是 \(B\) 的特征值，并写出对应的特征向量。 (4) 以上三个结论说明了相似变换的什么核心性质？

    答案提示： (1) \(\text{tr}(B) = \text{tr}(P^{-1}AP) = \text{tr}(APP^{-1}) = \text{tr}(A)\)。迹的循环不变性。(2) \(\det(B) = \det(P^{-1})\det(A)\det(P) = (1/\det(P))\cdot\det(A)\cdot\det(P) = \det(A)\)。(3) \(B(P^{-1}\mathbf{x}) = P^{-1}APP^{-1}\mathbf{x} = P^{-1}A\mathbf{x} = P^{-1}(\lambda\mathbf{x}) = \lambda(P^{-1}\mathbf{x})\)。所以 \(\lambda\) 也是 \(B\) 的特征值，对应特征向量 \(P^{-1}\mathbf{x}\)。(4) 相似变换不改变矩阵的谱（特征值）、迹和行列式——这些量是线性变换本身的固有属性，不依赖于「坐标系」的选择。这正是基变换的代数本质：换个基，矩阵「长相」变了，但「内在属性」不变。

    A3. 考虑一个两层的神经网络（无激活函数，纯线性，无偏置）： $\(\mathbf{y} = W_2(W_1 \mathbf{x}).\)$ 其中 \(W_1 \in \mathbb{R}^{3 \times 4}\)，\(W_2 \in \mathbb{R}^{2 \times 3}\)，\(\mathbf{x} \in \mathbb{R}^4\)，\(\mathbf{y} \in \mathbb{R}^2\)。

    (1) 该网络等效于一个什么样的单矩阵变换 \(W_{\text{eff}}\)？写出 \(W_{\text{eff}}\) 的维度。 (2) \(W_{\text{eff}}\) 的秩最大是多少？这说明了堆叠纯线性层的什么本质限制？ (3) 如果我们在两层之间插入一个 ReLU 激活：\(\mathbf{h} = \text{ReLU}(W_1\mathbf{x})\)，\(\mathbf{y} = W_2\mathbf{h}\)，还能把整个网络写成一个 \(W_{\text{eff}}\) 吗？为什么不能？

    答案提示： (1) \(\mathbf{y} = W_2W_1\mathbf{x} = W_{\text{eff}}\mathbf{x}\)，\(W_{\text{eff}} \in \mathbb{R}^{2 \times 4}\)。两个线性变换的复合仍是线性变换。(2) \(W_{\text{eff}}\) 的秩 \(\le \min(\text{rank}(W_1), \text{rank}(W_2)) \le \min(3, 2) = 2\)。这意味着无论 \(W_1\) 多大，最终「信息瓶颈」被 \(W_2\) 的输出维度 \(2\) 卡死了——堆纯线性层不增加网络的表达能力（表达能力 \(\le\) 最窄层的秩）。(3) 不能。ReLU 是分段线性的——它在输入空间的不同区域对应不同的「有效 \(W_{\text{eff}}\)」：当 \(W_1\mathbf{x}\) 的某些分量 \(\le 0\) 时，对应神经元「关掉」，\(W_2\) 中对应的列被清零。ReLU 把输入空间切成多个线性区域，每个区域里的 \(W_{\text{eff}}\) 不同——这就是深度网络的「非线性」来源。

    B 档¶

    B1. 为什么说 Embedding 层本质上是基变换？从线性代数角度解释：预训练学到的 Embedding 矩阵的列向量有什么几何含义？

    答案提示： Embedding 矩阵 \(W \in \mathbb{R}^{d \times |V|}\) 将 one-hot 向量（标准基）映射为稠密表示：\(\mathbf{e}_i = W \cdot \text{one\_hot}(i)\)。\(W\) 的每一列就是一个 token 的 \(d\) 维向量表示。从基变换角度看： - 标准基（one-hot）中每个 token 是彼此正交的，毫无语义关系； - 学到的 \(W\) 的列构成一组新的「语义基」——在这个基下，语义相近的 token 向量接近，语义关系变成向量运算（经典例子：\(\mathbf{e}_{\text{国王}} - \mathbf{e}_{\text{男人}} + \mathbf{e}_{\text{女人}} \approx \mathbf{e}_{\text{王后}}\)）。

    好的 Embedding = 找到一组基，使得语义方向在该基下线性可分。 这正是分布式表示（distributed representation）的线性代数本质。

    B2. 为什么 Layer Normalization 放在激活函数前面和后面效果不同？从投影几何的角度给出解释。

    答案提示： 设一层计算为 \(\mathbf{h} = \sigma(W\mathbf{x} + \mathbf{b})\)。

    • LN 放在激活前（Pre-LN，Transformer 中的标准做法）：标准化线性变换的输出 \(\mathbf{z} = W\mathbf{x} + \mathbf{b}\)，即 \(\text{LN}(\mathbf{z}) = \frac{\mathbf{z} - \mu}{\sigma} \odot \gamma + \beta\)。这是在仿射空间中做规范化——消除了不同样本间均值和方差的偏移（covariate shift），让激活函数接收到「规矩」的输入。
    • LN 放在激活后（Post-LN）：标准化的是非线性变换的结果 \(\sigma(\mathbf{z})\)。由于激活函数（如 ReLU）已经改变了分布的形状（截断负值、压缩或拉伸），此时再标准化相当于试图恢复被激活函数扭曲的统计特性——效果更差且训练更不稳定。

    从投影几何角度看，LN 将每个样本的特征向量投影到以原点为中心、半径为 \(\sqrt{d}\)（或经由 \(\gamma,\beta\) 仿射调整）的超球面上。Pre-LN 在进入非线性之前统一了所有样本的「尺度」，确保下一线性层接收到的数据分布稳定——这正是 LN 比 BatchNorm 更适合序列模型的原因：它不依赖 batch 统计量，逐样本独立做投影。

    B3. 考虑一个两层的 ReLU 网络，输入 \(\mathbf{x} \in \mathbb{R}^2\)，隐藏层有 \(3\) 个神经元： $\(\mathbf{h} = \text{ReLU}(W_1 \mathbf{x} + \mathbf{b}_1), \quad \hat{y} = W_2 \mathbf{h} + b_2.\)$ 其中 \(W_1 \in \mathbb{R}^{3 \times 2}\)。

    (1) 第一个隐藏神经元在什么条件下激活（输出 \(>0\)）？用 \(W_1\) 的行和 \(\mathbf{b}_1\) 的分量写出条件。在 \(\mathbb{R}^2\) 平面上，这个条件对应什么几何对象？ (2) 三个神经元各自定义了一条分界线。这些线把整个 \(\mathbb{R}^2\) 平面最多分成多少个区域？在每个区域内，网络的输出 \(\hat{y}\) 是什么形式？（提示：在某区域内哪些神经元激活是固定的） (3) 这如何说明「深度网络 = 分段线性函数」的直觉？更多的层和更宽的隐藏层如何增加分段数？

    答案提示： (1) 第 \(i\) 个神经元的激活条件：\(\mathbf{w}_i^{\mathsf T}\mathbf{x} + b_i > 0\)，其中 \(\mathbf{w}_i^{\mathsf T}\) 是 \(W_1\) 的第 \(i\) 行。在 \(\mathbb{R}^2\) 中，这是一条直线（超平面）的一侧——直线的方程是 \(\mathbf{w}_i^{\mathsf T}\mathbf{x} + b_i = 0\)。每个神经元把空间切成两半。

    (2) 3 条直线最多把平面分成 \(1 + 3 + 3 + 1 = 7\) 个区域（一般地，\(k\) 条直线最多分成 \(1 + k(k+1)/2\) 个区域，但这里直线可能不完全相交）。在每个区域内，哪些神经元激活是固定的（某些始终激活，某些始终不激活）→ \(\hat{y} = W_2^{\text{(该区域的激活模式)}} \cdot \mathbf{x} + \text{const}\) 是一个线性函数。从区域 A 走到区域 B 时，激活模式跳变 → \(\hat{y}\) 的线性函数也跳变。

    (3) 因此一个 ReLU 网络就是输入空间的一个分段线性函数。每一层的新超平面与之前的所有超平面相交，在它们的「折痕」上产生新的折痕——这创造了指数级的分段数。深层窄网络可以用很少的神经元产生极多的线性区域（深度 = 分段的指数放大器）；而单层宽网络只是简单地把空间切成均匀的块。

    B4. 手算反向传播。考虑一个极简的两层网络（无偏置，MSE 损失）： $\(\mathbf{h} = W_1 \mathbf{x}, \quad \hat{y} = W_2 \mathbf{h}, \quad L = \frac{1}{2}(\hat{y} - y)^2.\)$ 其中 \(W_1 = \begin{bmatrix} 1 & 2 \end{bmatrix}\)，\(W_2 = \begin{bmatrix} 3 \end{bmatrix}\)，\(\mathbf{x} = (1, 1)^{\mathsf T}\)，\(y = 5\)。

    (1) 手算前向传播：\(\mathbf{h}\) 和 \(\hat{y}\) 各是多少？\(L\) 是多少？ (2) 手算 \(\frac{\partial L}{\partial \hat{y}}\)，然后用链式法则反推 \(\frac{\partial L}{\partial W_2}\) 和 \(\frac{\partial L}{\partial W_1}\)。 (3) 用 PyTorch 验证你的手算结果（写两行伪代码即可）。

    答案提示： (1) \(\mathbf{h} = W_1\mathbf{x} = [1,2]\cdot[1,1]^{\mathsf T} = 3\)。\(\hat{y} = W_2 \cdot 3 = 9\)。\(L = \frac{1}{2}(9-5)^2 = 8\)。

    (2) \(\frac{\partial L}{\partial \hat{y}} = \hat{y} - y = 4\)。 \(\frac{\partial L}{\partial W_2} = \frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial W_2} = 4 \cdot \mathbf{h} = 4 \cdot 3 = 12\)。 \(\frac{\partial L}{\partial \mathbf{h}} = \frac{\partial L}{\partial \hat{y}} \cdot W_2^{\mathsf T} = 4 \cdot 3 = 12\)。 \(\frac{\partial L}{\partial W_1} = \frac{\partial L}{\partial \mathbf{h}} \cdot \mathbf{x}^{\mathsf T} = 12 \cdot [1, 1] = \begin{bmatrix} 12 & 12 \end{bmatrix}\)。

    直观检查：\(\hat{y}\) 比 \(y\) 大 \(4\)（误差 \(=4\)）。\(W_2=3\) 把 \(\mathbf{h}\) 放大了 3 倍，所以梯度反向流过 \(W_2\) 时也被放大 3 倍：\(\frac{\partial L}{\partial \mathbf{h}} = 12\)。\(\frac{\partial L}{\partial W_1}\) 的两个分量都是 \(12\)，因为 \(\mathbf{x}=(1,1)\) 的两个特征贡献相同。

    (3) 伪代码： 

    import torch
W1 = torch.tensor([[1., 2.]], requires_grad=True)
W2 = torch.tensor([[3.]], requires_grad=True)
x = torch.tensor([1., 1.]).unsqueeze(1)
y = torch.tensor([5.])
h = W1 @ x; y_hat = W2 @ h; loss = 0.5 * (y_hat - y).pow(2).sum()
loss.backward()
print(W1.grad, W2.grad)  # tensor([[12., 12.]])  tensor([[12.]])

    B5. 以下场景中，PCA 本质上在做一种「基变换」。请为每个场景说明：① 原始基是什么？② PCA 找到的新基由什么向量构成？③ 这个新基的哪几个维度通常被保留？

    (1) 一个 \(N \times 768\) 的词嵌入矩阵，想降到 256 维。 (2) LoRA 对大语言模型的权重更新 \(W \in \mathbb{R}^{d \times d}\) 做低秩分解 \(W \approx BA\)（\(B \in \mathbb{R}^{d \times r}\)，\(A \in \mathbb{R}^{r \times d}\)，\(r \ll d\)）。 (3) 图像去噪：对一批 \(28 \times 28 = 784\) 维的图像向量做 PCA，保留前 \(k\) 个主成分后重建。

    答案提示： (1) ① 原始基 = 标准基（one-hot），每个 token 是彼此正交的维数 \(|V|\) 向量。② PCA 找到的新基 = 协方差矩阵 \(X^{\mathsf T}X\) 的前 \(k\) 个特征向量——即词向量分布方差最大的 \(k\) 个「语义方向」。③ 保留前 256 个主成分（对应最大特征值的方向）。

    (2) ① 原始基 = 权重空间 \(\mathbb{R}^{d \times d}\) 的标准基（每个参数一个独立维度）。② LoRA 不做显式的 PCA，但其思想与 PCA 同源：假设权重更新矩阵 \(W\) 是低秩的——它只在少数几个「重要方向」上有显著信息，其余方向接近 \(0\)。\(B\) 的 \(r\) 列就是这 \(r\) 个方向的基向量，\(A\) 的每行是对应的「坐标」。③ LoRA 保留前 \(r\) 个方向（\(r=8\sim64\)），丢弃 \(d-r\) 个方向。这等价于假设：大模型微调时只需要在极低维子空间内更新参数——这和 PCA 截断维度的逻辑如出一辙。

    (3) ① 原始基 = 784 个像素上的标准基（每个像素一个维度）。② PCA 新基 = 协方差矩阵的特征向量，即「特征脸」（eigenfaces）——它们不是独立的像素，而是像素之间的「相关模式」。③ 保留前 \(k\) 个特征脸（\(k \ll 784\)），认为噪声集中在低方差方向（小特征值方向）。重建时只投向这 \(k\) 个方向，天然滤掉了噪声。
