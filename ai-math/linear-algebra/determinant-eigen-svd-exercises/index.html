    习题与思考：行列式、特征值与 SVD¶

    A 档巩固基础，B 档培养 ML 直觉。答案提示见各题末尾。

    三、行列式、特征值与 SVD（§9–§13）¶

    A 档¶

    A1. 化二次型 \(f(x_1,x_2)=3x_1^2+2x_1x_2+3x_2^2\) 为标准形，并判断其正定性。

    答案提示： 对称矩阵 \(A = \begin{bmatrix}3&1\\1&3\end{bmatrix}\)。特征方程 \(|\lambda I-A| = (\lambda-3)^2-1=0\)，特征值 \(\lambda_1=4\)，\(\lambda_2=2\)。标准形：\(f = 4y_1^2 + 2y_2^2\)。所有特征值 \(>0\)，正定。几何上，这是一个「拉长的碗」——等高线是同心椭圆，长短轴方向由特征向量给出。

    A2. 求 \(A = \begin{bmatrix}3&1\\1&3\end{bmatrix}\) 的特征值与特征向量，写出对角化形式 \(A = Q\Lambda Q^{\mathsf T}\)。

    答案提示： \(\lambda_1=4\)，\(\mathbf{v}_1 = (1,1)^{\mathsf T}/\sqrt{2}\)；\(\lambda_2=2\)，\(\mathbf{v}_2 = (-1,1)^{\mathsf T}/\sqrt{2}\)。\(Q = \begin{bmatrix}1/\sqrt{2} & -1/\sqrt{2} \\ 1/\sqrt{2} & 1/\sqrt{2}\end{bmatrix}\)（正交矩阵，\(Q^{\mathsf T}=Q^{-1}\)），\(\Lambda = \text{diag}(4,2)\)。验证：\(Q\Lambda Q^{\mathsf T} = A\)。

    A3. 用矩阵微积分求梯度。设 \(L(W) = \frac{1}{2}\|WX - Y\|_F^2\)，其中 \(X \in \mathbb{R}^{n \times m}\) 为输入矩阵，\(Y \in \mathbb{R}^{d \times m}\) 为目标矩阵。求 \(\nabla_W L\)。

    答案提示： 展开：

    \[L = \frac{1}{2}\text{tr}\big((WX-Y)^{\mathsf T}(WX-Y)\big).\]

    对 \(W\) 求微分：\(dL = \text{tr}\big((WX-Y)^{\mathsf T} \cdot dW \cdot X\big) = \text{tr}\big(X(WX-Y)^{\mathsf T} \cdot dW\big)\)。因此：

    \[\nabla_W L = (WX - Y)X^{\mathsf T}.\]

    这就是线性层反向传播中权重梯度的矩阵形式——它告诉你梯度是误差矩阵右乘输入的转置。

    A4. 求 softmax 函数 \(\mathbf{s}(\mathbf{z}) = \frac{e^{z_i}}{\sum_j e^{z_j}}\) 的雅可比矩阵 \(J = \partial \mathbf{s} / \partial \mathbf{z}\)。

    答案提示： 当 \(i \neq j\)：

    \[\frac{\partial s_i}{\partial z_j} = -s_i s_j.\]

    当 \(i = j\)：

    \[\frac{\partial s_i}{\partial z_i} = s_i(1-s_i).\]

    矩阵形式：

    \[J = \text{diag}(\mathbf{s}) - \mathbf{s}\mathbf{s}^{\mathsf T}.\]

    这个 \(J\) 是对称且半正定的（秩 \(=K-1\)，零空间由 \(\mathbf{1}\) 张成）。它与交叉熵配合使用时会产生优雅的简化——见下一题。

    A5. 不直接计算行列式，仅用行列式的性质判断以下命题的真伪：

    (1) 若 \(A\) 的某一行全是 \(0\)，则 \(\det(A)=0\)。 (2) 若 \(A\) 的两行成比例，则 \(\det(A)=0\)。 (3) 若 \(\det(A)=2\)，则 \(\det(A^{-1}) = -2\)。 (4) 若对 \(A\) 做初等行变换 \(R_2 \leftarrow R_2 - 3R_1\)，行列式的值乘以 \(-3\)。 (5) 若 \(\det(A - \lambda I)=0\)，则 \(\lambda\) 是 \(A\) 的特征值——等价于 \(A\) 将某个非零向量拉伸 \(\lambda\) 倍。

    答案提示： (1) 对。对那一行做展开（按该行展开），每项都含 \(0\) 因子。(2) 对。成比例意味着行线性相关，高斯消元会产生零行。(3) 错。\(\det(A^{-1}) = 1/\det(A) = 1/2\)，不是 \(-2\)。行列式的倒数，不是相反数。(4) 错。\(R_2 \leftarrow R_2 - 3R_1\) 这种「把一行的倍数加到另一行」不改变行列式。只有交换两行才会变号，只有对一行乘以 \(c\) 才会把行列式乘以 \(c\)。(5) 对。这就是特征方程的定义：\(\det(A - \lambda I)=0 \iff A\mathbf{x} = \lambda\mathbf{x}\) 有非零解。

    B 档¶

    B1. 某深度网络的 Hessian 特征值为 \(\{125,\; 3.2,\; 0.01,\; -0.5,\; -18\}\)。判断当前点在优化地形中的位置（极小点？鞍点？），并预测 SGD 和 Adam 各自的表现差异。

    答案提示： 同时存在正特征值和负特征值 \(\Rightarrow\) 当前点是一个鞍点（saddle point），而非局部极小。正惯性指数 \(p=3\)，负惯性指数 \(q=2\)。条件数 \(\kappa \approx 125/0.01 = 12500\)（极差）。

    SGD： 在特征值 \(125\) 的方向上剧烈震荡（步长稍大即 overshoot），在特征值 \(0.01\) 的方向上几乎寸步难行；同时沿两个负特征值 \(-0.5\) 和 \(-18\) 的方向逃离鞍点——这在 SGD 中可能表现为 loss 在鞍点附近「卡住」很久后突然继续下降。Adam： 自适应地为每个方向调整有效步长——强曲率方向用微小步、弱曲率方向用大步、负曲率方向也以合适步长下行——\(3\) 个正方向平稳下降、\(2\) 个负方向快速逃离鞍点，远优于 SGD。

    B2. 证明恒等式 \(\nabla_{\mathbf{z}}\;\text{CrossEntropy}(\text{softmax}(\mathbf{z}), \mathbf{y}) = \hat{\mathbf{y}} - \mathbf{y}\)（使用微分法）。这是机器学习中最优雅的梯度公式之一——不论用什么模型，只要输出端是 softmax + 交叉熵，最后一层的梯度永远如此简洁。

    答案提示： 设 \(\hat{\mathbf{y}} = \text{softmax}(\mathbf{z})\)，损失 \(L = -\sum_k y_k \log \hat{y}_k\)（\(y\) 为 one-hot 或概率向量）。

    微分：\(dL = -\sum_k \frac{y_k}{\hat{y}_k} d\hat{y}_k = -\mathbf{y}^{\mathsf T} \text{diag}(\hat{\mathbf{y}})^{-1} \, d\hat{\mathbf{y}}\)。

    由上题 \(d\hat{\mathbf{y}} = (\text{diag}(\hat{\mathbf{y}}) - \hat{\mathbf{y}}\hat{\mathbf{y}}^{\mathsf T}) d\mathbf{z}\)。

    代入：

    \[dL = -\mathbf{y}^{\mathsf T} \text{diag}(\hat{\mathbf{y}})^{-1} (\text{diag}(\hat{\mathbf{y}}) - \hat{\mathbf{y}}\hat{\mathbf{y}}^{\mathsf T}) d\mathbf{z} = -\mathbf{y}^{\mathsf T}(I - \mathbf{1}\hat{\mathbf{y}}^{\mathsf T}) d\mathbf{z}.\]

    由于 \(\sum_k y_k = 1\)，\(-\mathbf{y}^{\mathsf T} \mathbf{1}\hat{\mathbf{y}}^{\mathsf T} = -\hat{\mathbf{y}}^{\mathsf T}\)，因此 \(dL = (\hat{\mathbf{y}} - \mathbf{y})^{\mathsf T} d\mathbf{z}\)。故而：

    \[\nabla_{\mathbf{z}} L = \hat{\mathbf{y}} - \mathbf{y}.\]

    这个公式意味着反向传播中最后一层的「误差信号」就是预测概率减去真实标签——极其简洁，且数值稳定。

    B3. 为什么协方差矩阵必须是半正定的？如果样本协方差矩阵在计算中出现了负特征值（可能由于浮点误差），会有什么实际影响？如何修复？

    答案提示： 对任意向量 \(\mathbf{x}\)，

    \[\mathbf{x}^{\mathsf T}\Sigma\mathbf{x} = \mathbf{x}^{\mathsf T}\mathbb{E}[(\mathbf{X}-\boldsymbol{\mu})(\mathbf{X}-\boldsymbol{\mu})^{\mathsf T}]\mathbf{x} = \mathbb{E}[(\mathbf{x}^{\mathsf T}(\mathbf{X}-\boldsymbol{\mu}))^2] = \text{Var}(\mathbf{x}^{\mathsf T}\mathbf{X}) \ge 0,\]

    故 \(\Sigma\) 半正定是协方差矩阵的数学必然。

    实际中，若因浮点舍入误差或样本量不足导致负特征值出现： - 概率密度函数在对应方向上为负——无意义； - Cholesky 分解失败（要求正定），多元正态采样崩溃； - 马氏距离 \(\sqrt{(\mathbf{x}-\boldsymbol{\mu})^{\mathsf T}\Sigma^{-1}(\mathbf{x}-\boldsymbol{\mu})}\) 失去几何解释。

    解决方案： shrinkage（收缩估计），\(\Sigma \leftarrow (1-\alpha)\Sigma + \alpha I\)，用一个小 \(\alpha\)（如 \(0.01\)）将所有特征值抬升 \(\alpha\)，强制最小特征值为正。

    B4. 矩阵 \(A = \begin{bmatrix}3&0\\0&1\end{bmatrix}\) 的 SVD 是什么？不用算——它本身就已经是一个「准 SVD」了。但有更一般的情况：设 \(A\) 的 SVD 为 \(A = U\Sigma V^{\mathsf T}\)，其中 \(\Sigma = \text{diag}(5, 2, 0)\)。

    (1) \(A\) 的秩是多少？ (2) 描述 \(A\) 对 \(\mathbb{R}^3\) 中的单位球做了什么几何变换（用 \(V^{\mathsf T}\)、\(\Sigma\)、\(U\) 三步拆解）。 (3) \(\mathbf{v}_3\)（\(V\) 的第 3 列）对应奇异值 \(0\)。\(A\) 把 \(\mathbf{v}_3\) 映射到了哪里？这说明 \(\mathbf{v}_3\) 在什么子空间里？ (4) （ML 扩展）PCA 本质上是对协方差矩阵做 SVD。如果 PCA 后发现最小的若干个奇异值接近 \(0\)，你对数据有什么判断？对下游模型设计有何指导？

    答案提示： (1) 秩 \(=\) 非零奇异值的个数 \(=2\)。

    (2) 三步拆解： ① \(V^{\mathsf T}\) 是正交矩阵 → 旋转/反射单位球，形状不变。② \(\Sigma\) 把球映射成一个椭球：\(x\) 轴方向拉伸 \(5\times\)，\(y\) 轴方向拉伸 \(2\times\)，\(z\) 轴方向压扁到 \(0\)（坍缩！）。球变成 \(xy\) 平面内的一个椭圆盘。③ \(U\) 把这个椭圆盘旋转/反射到输出空间，不改变形状。

    (3) \(A\mathbf{v}_3 = U\Sigma V^{\mathsf T}\mathbf{v}_3 = U\Sigma (0,0,1)^{\mathsf T} = U(0,0,0)^{\mathsf T} = \mathbf{0}\)。\(\mathbf{v}_3\) 被 \(A\) 映射为零向量 → \(\mathbf{v}_3 \in N(A)\)（零空间）。SVD 的神奇之处：\(V\) 的最后 \(n-r\) 列就是零空间的标准正交基。

    (4) 奇异值接近 \(0\) → 协方差矩阵有接近 \(0\) 的特征值 → 数据几乎分布在一个低维子空间内，沿某些方向的方差近乎 \(0\)。这意味着： - 输入维度假高——可以大幅降维而不损失信息； - 模型容量不必正比于名义维度——LoRA（低秩适配）的核心洞察：大模型的权重更新也可以做低秩约束，因为问题本身就是低秩的； - 如果强行在高维空间建模，最小二乘/线性回归的系数会被噪声放大（条件数问题），需要正则化。
