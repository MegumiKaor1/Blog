    线性代数¶

    为什么 AI 要学线性代数？

    因为神经网络里的每一层本质上就是一个线性变换（矩阵乘法）再加一个非线性激活。你的输入是一串数字（向量），权重是矩阵，输出是变换后的向量。不理解线性代数，你就永远看不懂 nn.Linear(768, 3072) 在做什么。

    Gilbert Strang 说得好：「太多学生知道怎么乘矩阵，但不知道为什么要乘。我想改变这一点。」

    学习资源总览¶

    线性代数最美的部分，藏在三种互补的视角里。建议把它们结合起来学：

    资源 视角 适合谁
    Strang《Introduction to Linear Algebra》第 6 版 结构主义：从方程组到子空间到 SVD，一条逻辑主线 想要系统性理解「线性代数到底是什么」的人
    MIT 18.06 公开课（34 讲，Strang 亲授） 同上的视频版，Strang 在黑板前如讲故事般展开 看书累的时候看视频，Strang 的魅力无可替代
    3Blue1Brown「线性代数的本质」 几何直觉：把每一个概念「可视化」 公式看晕了就看它，看完再回头看公式会豁然开朗

    我的建议：先花一个下午看完 3Blue1Brown 的全系列（约 3 小时），建立几何直觉，然后再啃 Strang 的书。顺序反过来也可以——两种路线都有人走通。

    Strang 的大图景：线性代数的三层理解¶

    Strang 把整个线性代数组织成三个核心方程，一个比一个「更对角化」：

    层次 方程 回答的问题 在 ML 中的影子
    第一层 \(Ax = b\) 给定输入 \(b\)，能解出 \(x\) 吗？有唯一解吗？ 神经网络的每一层前向传播
    第二层 \(Ax = \lambda x\) 矩阵作用在哪些方向上只是拉伸？拉伸了多少？ PCA 降维、谱聚类、图卷积
    第三层 \(Av = \sigma u\) 能不能找到两组正交基，让矩阵变成纯对角？ SVD、LoRA、推荐系统

    这三层不是独立的——它们是逐步揭示矩阵结构的递进过程。Strang 把 SVD 称为「线性代数的皇冠上的明珠」，因为它是最终的、完美的对角化，每一个矩阵都有 SVD。

    核心概念体系¶

    1. 向量与线性组合（一切的起点）¶

    Strang 讲课的第一句话就是：「线性代数的核心是线性组合。」

    给定向量 \(v_1, v_2, \dots, v_k\)，它们的线性组合是： $\(c_1 v_1 + c_2 v_2 + \dots + c_k v_k\)$

    两种视角看线性方程组 \(Ax = b\)（Strang Lecture 1 的精髓）：

    行图景 (Row Picture) 列图景 (Column Picture)
    看什么 每个方程是一条线/平面，求交点 \(b\) 是 \(A\) 各列的线性组合，求组合系数
    问什么 这些线/平面交于哪一点？ 用 \(A\) 的列怎么拼出 \(b\)？
    直觉 从方程组角度理解 理解矩阵乘法的正确方式

    列图景是理解一切的钥匙：\(Ax\) 就是「\(A\) 的列的线性组合，组合系数是 \(x\) 的各分量」。这也是神经网络前向传播 \(y = Wx\) 的几何本质。

    3Blue1Brown 对此有一个漂亮的动画：矩阵就是一个空间变换——它把标准基向量 \(\hat{i}, \hat{j}\) 挪到新的位置，然后把整个空间跟着一起「拉扯」。

    2. 消元与 \(A = LU\) 分解¶

    Gaussian 消元不是「操作步骤」，而是矩阵分解。

    消元的过程等价于用一个下三角矩阵 \(L\) 去乘 \(A\)，得到上三角矩阵 \(U\)： $\(A = LU\)$

    • \(L\)（Lower triangular）的对角线全是 1，非零元是消元乘数
    • \(U\)（Upper triangular）是消元后的结果，主元在对角线上

    这为什么重要？因为 解 \(Ax = b\) 变成了两步代入：先解 \(Lc = b\)（前向代入，\(O(n^2)\)），再解 \(Ux = c\)（回代，\(O(n^2)\)）。比直接求逆快得多，也是所有数值线性代数的起点。

    3. 四个基本子空间（Strang 的皇冠）¶

    这是 Strang 对整个学科最大的教学贡献。每个 \(m \times n\) 矩阵 \(A\)（秩为 \(r\)）定义了四个子空间：

    子空间 符号 在哪个空间 维数 含义
    列空间 \(C(A)\) \(\mathbb{R}^m\) \(r\) \(A\) 的列能张成的所有向量
    零空间 \(N(A)\) \(\mathbb{R}^n\) \(n-r\) 被 \(A\) 映射到 0 的所有 \(x\)
    行空间 \(C(A^T)\) \(\mathbb{R}^n\) \(r\) \(A\) 的行能张成的所有向量
    左零空间 \(N(A^T)\) \(\mathbb{R}^m\) \(m-r\) 满足 \(y^T A = 0\) 的所有 \(y\)

    线性代数的基本定理（第一部分）：

    • 行空间 \(\perp\) 零空间，且它们互补地张成整个 \(\mathbb{R}^n\)
    • 列空间 \(\perp\) 左零空间，且它们互补地张成整个 \(\mathbb{R}^m\)

    这个定理的美妙之处在于：它告诉你 \(Ax = b\) 什么时候有解、解的「结构」是什么： $\(x = x_{\text{particular}} + x_{\text{nullspace}}\)$

    一个特解 + 零空间里的任意向量 = 全部解。这在 ML 里直接对应：一个过参数化的模型有无数个解，但正则化帮你从零空间里挑一个。

    4. 正交性与投影¶

    当 \(b\) 不在 \(A\) 的列空间里时，\(Ax = b\) 无解。这时我们退而求其次：找 \(\hat{x}\) 让 \(A\hat{x}\) 尽可能接近 \(b\)。

    这就是最小二乘法——把 \(b\) 投影到列空间 \(C(A)\) 上： $\(A\hat{x} = A(A^T A)^{-1} A^T b\)$

    投影矩阵 \(P = A(A^T A)^{-1} A^T\) 满足 \(P^2 = P\)（投影两次等于投影一次）且 \(P^T = P\)（对称）。

    在 ML 里：线性回归的解析解就是这个公式。更广义地说，任何「找最优近似」的问题——从 PCA 到自编码器——本质上都是某种投影。

    5. 行列式：体积的度量¶

    Strang 用 10 条性质来定义行列式（而不是从 cofactor 公式开始），核心直觉：

    \(\det(A)\) 是 \(A\) 的列向量张成的平行多面体的「有向体积」。

    • \(\det(A) = 0\) 意味着列向量线性相关，张成的体积为 0——矩阵不可逆
    • \(\det(AB) = \det(A) \det(B)\)：先变换 \(B\) 再变换 \(A\)，体积缩放因子相乘

    3Blue1Brown 的可视化让这条直觉极其清晰：线性变换把单位正方形变成一个平行四边形，它的面积就是行列式的绝对值，正负号表示是否翻转了空间。

    6. 特征值与特征向量（第二层）¶

    \[Ax = \lambda x\]

    矩阵 \(A\) 在某些特殊方向 \(x\) 上，作用效果仅仅是拉伸，拉伸倍数就是特征值 \(\lambda\)。

    对角化：如果 \(A\) 有 \(n\) 个线性独立的特征向量，则 $\(A = S \Lambda S^{-1}\)$

    其中 \(S\) 的列是特征向量，\(\Lambda\) 是对角矩阵（对角元 = 特征值）。这意味着在特征向量构成的坐标系下，\(A\) 就是一个纯拉伸。

    对称矩阵在 ML 中最常见（协方差矩阵、Hessian、Gram 矩阵），它们有完美的性质： - 特征值全是实数 - 特征向量可以选为正交的 - 谱分解：\(A = Q \Lambda Q^T\)

    正定矩阵（所有特征值 \(> 0\)）是 ML 中的常客：Hessian 正定意味着到了局部极小点，协方差矩阵天然正定。

    7. 奇异值分解 SVD（第三层，皇冠上的明珠）¶

    每一个 \(m \times n\) 矩阵 \(A\)（秩为 \(r\)）都可以分解为： $\(A = U \Sigma V^T\)$

    • \(U\)（\(m \times m\)）：左奇异向量，\(C(A)\) 和 \(N(A^T)\) 的标准正交基
    • \(\Sigma\)（\(m \times n\)）：对角矩阵，\(\sigma_1 \ge \sigma_2 \ge \dots \ge \sigma_r > 0\)
    • \(V\)（\(n \times n\)）：右奇异向量，\(C(A^T)\) 和 \(N(A)\) 的标准正交基

    SVD 为什么是皇冠上的明珠？

    1. 普适性：不要求方阵，不要求可对角化——任何矩阵通吃
    2. 几何意义清晰：\(A\) = 旋转（\(V^T\)）→ 拉伸（\(\Sigma\)）→ 旋转（\(U\)）
    3. 四个子空间一起给：\(U\) 的前 \(r\) 列是 \(C(A)\) 的基，后 \(m-r\) 列是 \(N(A^T)\) 的基；\(V\) 的前 \(r\) 列是 \(C(A^T)\) 的基，后 \(n-r\) 列是 \(N(A)\) 的基

    SVD 与 PCA 的等价性：对中心化数据 \(X\)，其协方差矩阵的特征向量 = \(X\) 的右奇异向量。PCA 本质上就是截断 SVD。

    Strang 在 2023 年第 6 版中大幅扩展了 SVD 章节，加入了 PCA、图像压缩、深度学习连接——因为 SVD 正是数据科学的数学引擎。

    8. 线性变换与基变换¶

    线性代数的「现代化」视角：矩阵是线性变换在特定基下的表示。

    同一个线性变换 \(T\)，在不同基下的矩阵表示通过相似变换关联： $\(B = P^{-1} A P\)$

    这意味着很多看似不同的矩阵在本质上代表同一个变换，只是观察的「角度」不同。Strang 会在课程后半段反复强调这一点，把前面所有概念串起来。

    在 ML 中：嵌入层本质上是在做基变换——把离散的 token 映射到一个连续的向量空间里，选择好的「基」（预训练嵌入）比从零开始重要得多。

    在 ML/AI 中的具体应用¶

    神经网络的前向传播 = 一系列线性变换的复合¶

    y = W_3 @ relu(W_2 @ relu(W_1 @ x + b_1) + b_2) + b_3

    每一层 W @ x + b 是一个仿射变换（线性变换 + 平移）。整个网络是一个「分段线性函数」——ReLU 把空间切割成不同的线性区域，在每个区域内网络都是线性的。

    PCA = 截断 SVD¶

    给定数据矩阵 \(X\)（已中心化），PCA 找方差最大的方向： 1. 对 \(X\) 做 SVD：\(X = U \Sigma V^T\) 2. 取前 \(k\) 个奇异值对应的右奇异向量作为主成分 3. 降维：\(X_k = U_k \Sigma_k V_k^T\)

    这就是 Eckart-Young 定理：截断 SVD 给出了在 Frobenius 范数下矩阵的最佳低秩近似。

    LoRA（Low-Rank Adaptation）= \(A + \Delta W = A + BA\)¶

    LoRA 的核心思想：微调时不去修改原权重 \(W\)，而是加一个低秩增量 \(\Delta W = BA\)： - \(B\) 是 \(d \times r\) 的矩阵 - \(A\) 是 \(r \times k\) 的矩阵 - \(r\) 很小（比如 8 或 16）

    这等价于：SVD 告诉我们最重要的信息集中在前几个奇异值上，微调只需要调整矩阵的「低秩部分」。

    Attention 中的矩阵乘法¶

    自注意力的核心运算： $\(\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right) V\)$

    • \(QK^T\)：\(n \times d\) 乘 \(d \times n\)，得到 \(n \times n\) 的注意力矩阵——每一对 token 之间的内积
    • \(W^O \cdot \text{Concat}(\text{head}_1, \dots, \text{head}_h)\)：多头注意力的输出投影

    更多应用速查¶

    概念 在 ML 中的位置
    向量内积 注意力分数（\(Q \cdot K\)）、相似度计算（cosine similarity）
    矩阵乘法 y = Wx + b，每一层前向传播
    范数 L1/L2 正则化、梯度裁剪、权重衰减
    条件数 梯度爆炸/消失的数学根源，优化难度指标
    特征值 PCA 降维、谱聚类、图神经网络的谱方法
    SVD 推荐系统（矩阵分解）、LoRA、图像压缩
    伪逆 线性回归的解析解、过参数化模型的最小范数解
    张量 CNN 的多维特征图、多头注意力的批量运算

    💭 个人理解¶

    Strang 的方法好在哪里¶

    大部分线性代数教科书是「定义→定理→证明→例题」的流水线。Strang 反其道而行之：先让你看见、先建立直觉、先问「这有什么用」，再给形式化定义。

    他反复强调的「列图景」——把矩阵乘法看作列的线性组合——是我学线性代数多年后才真正内化的视角。一旦你习惯了这个视角，你会发现神经网络的前向传播、Attention 的加权求和、PCA 的投影，本质上都是同一件事的不同面目。

    3Blue1Brown 为什么不可或缺¶

    Strang 给了你结构和严谨，但 3Blue1Brown 给了你画面。他把「矩阵是一个空间变换」「行列式是面积缩放因子」「特征向量是不改变方向的轴」这些抽象概念做成了动画。有些东西你看十遍公式都不一定能理解，看一遍动画就懂了。两者的结合是学习线性代数的最佳方式。

    线性代数的「艺术」¶

    线性代数的美不在于算得快，而在于它用极少的几个核心概念——向量、矩阵、子空间、特征值——编织出了一整套理解「变换」和「结构」的语言。

    我认为最美的几个瞬间：

    1. 基本定理的对称性：四个子空间两两正交互补，SVD 一次性给出所有四个子空间的标准正交基。这种完美的对偶性像一首对称的诗。

    2. \(A = CR\) 分解的纯粹性：把一个秩为 \(r\) 的矩阵拆成 \(r\) 个独立的列 × \(r\) 个独立的行。没有多余信息，没有数值噪音——这就是秩的终极含义。

    3. Eckart-Young 的优雅：截断 SVD 给出的低秩近似在所有可能的低秩近似中是最优的。而这个证明简洁到只有几行。LoRA 能用、PCA 能用、推荐系统能用——因为它们背后是同一个定理。

    4. 学科的自我指涉：用线性代数来教线性代数——Strang 的四个子空间大图就是一张「概念关系图」，把整个学科的知识结构本身变成了一个向量空间。这种元层次的优雅让人着迷。

    学完线性代数之后¶

    你会发现 AI 不再是黑箱。梯度下降是在高维空间里沿着曲面滚下，Attention 是在度量向量之间的夹角，LoRA 是在低维流形上微调权重。一切都有几何意义。

    Strang 在他的书里写道：「线性代数是数据的数学。」在这个数据驱动的时代，这句话从未如此贴切。
