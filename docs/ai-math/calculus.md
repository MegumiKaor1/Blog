# 微积分

> **为什么 AI 要学微积分？**
>
> 因为**训练神经网络 = 求梯度**。梯度是导数在高维的推广，而导数来自微积分。
> 反向传播（Backpropagation）就是链式法则的一次大规模应用。
> 不理解微积分，你就看不懂 `loss.backward()` 到底干了什么。

## 在 AI 中的实际应用

| 概念 | 在 ML 中的位置 |
|------|---------------|
| **导数 / 偏导数** | 梯度下降的数学基础：参数往负梯度方向更新 |
| **链式法则** | 反向传播的核心——从 loss 逐层回传梯度 |
| **梯度** | 参数更新的方向和大小 |
| **Hessian 矩阵** | 二阶优化方法（Newton 法）、损失景观分析 |
| **积分** | 连续概率分布归一化、ELBO 推导中的边际化 |
| **Jacobian 矩阵** | Normalizing Flows 中的变量替换 |

## 学习路线

1. 极限、导数、偏导数
2. 链式法则
3. 梯度与方向导数
4. Hessian 矩阵与二阶优化
5. 积分基础（为概率论铺垫）
