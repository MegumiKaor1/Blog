# 信息论

> **为什么 AI 要学信息论？**
>
> 因为你每天在用的**交叉熵损失函数**，本质上来自信息论。
> VAE 的 ELBO 里每一项都是信息论概念。
> 扩散模型用熵和 KL 散度来度量生成质量。这是损失函数的「哲学基础」。

## 在 AI 中的实际应用

| 概念 | 在 ML 中的位置 |
|------|---------------|
| **熵 (Entropy)** | 衡量分布的不确定性，最大熵模型 |
| **交叉熵 (Cross Entropy)** | 分类任务的标准损失函数 |
| **KL 散度** | VAE 训练、知识蒸馏、扩散模型 ELBO |
| **互信息 (Mutual Information)** | 特征选择、InfoGAN、对比学习 (InfoNCE) |
| **熵正则化** | RL 中的探索激励、soft actor-critic |

## 学习路线

1. 自信息与熵
2. 交叉熵与 KL 散度
3. 互信息
4. 熵在 ML 损失函数中的角色
5. 熵正则化与变分推断
