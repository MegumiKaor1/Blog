# 线性代数

> **为什么 AI 要学线性代数？**
>
> 因为神经网络里的**每一层本质上就是一个线性变换**（矩阵乘法）再加一个非线性激活。
> 你的输入是一串数字（向量），权重是矩阵，输出是变换后的向量。
> 不理解线性代数，你就永远看不懂 `nn.Linear(768, 3072)` 在做什么。

## 在 AI 中的实际应用

| 概念 | 在 ML 中的位置 |
|------|---------------|
| **向量** | 词嵌入（word embedding）、图像展平为一维向量、隐层表示 |
| **矩阵乘法** | `y = Wx + b`——神经网络的每一层 |
| **特征值与特征向量** | PCA 降维、谱聚类、图神经网络 |
| **奇异值分解 (SVD)** | 推荐系统（矩阵分解）、LoRA 低秩微调 |
| **范数** | L1/L2 正则化、梯度裁剪 |
| **张量** | CNN 中的多维特征图、多头注意力中的批量矩阵乘法 |

## 学习路线

1. 向量、矩阵、张量的基本运算
2. 线性变换与基变换
3. 特征值与特征向量
4. 奇异值分解 (SVD)
5. 范数与内积空间
