# 最优化理论

> **为什么 AI 要学最优化理论？**
>
> 因为训练模型 = 在几百万维的参数空间中找一个最低点。
> Adam/AdamW 为什么比 SGD 好？为什么有时 loss 震荡下不去？
> 这些问题的答案都在最优化理论里。

## 在 AI 中的实际应用

| 概念 | 在 ML 中的位置 |
|------|---------------|
| **凸优化** | 理论上保证收敛的唯一框架 |
| **梯度下降** | 最基本的优化算法 |
| **动量 (Momentum)** | SGD with Momentum → 加速收敛、减少震荡 |
| **自适应学习率** | Adam/AdamW 的核心——每个参数有自己的学习率 |
| **约束优化** | 带约束的模型训练、对抗训练 |
| **收敛性分析** | 证明你的训练过程最终会到达一个解 |

## 学习路线

1. 凸集、凸函数、凸优化问题
2. 梯度下降及其收敛性
3. 动量方法
4. 自适应方法（AdaGrad → RMSProp → Adam）
5. 约束优化与 Lagrange 乘子法
