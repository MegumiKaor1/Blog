# 概率论与数理统计

> **为什么 AI 要学概率论？**
>
> 因为机器学习本质上是在做概率建模。
> 你希望模型输出「这张图是猫的概率」，而不是一个确定的数字。
> 最大似然估计、贝叶斯推断、VAE 的 ELBO 推导——全在概率框架下运行。

## 在 AI 中的实际应用

| 概念 | 在 ML 中的位置 |
|------|---------------|
| **随机变量与分布** | 建模数据分布 $p(x)$、潜变量分布 $p(z)$ |
| **条件概率** | 分类器 $p(y|x)$、生成模型 $p(x|z)$ |
| **贝叶斯公式** | 贝叶斯神经网络、变分推断 |
| **期望与方差** | 损失函数（期望风险）、BatchNorm |
| **最大似然估计 (MLE)** | 几乎所有生成模型的训练目标 |
| **KL 散度** | VAE 训练、知识蒸馏、扩散模型中的 ELBO |
| **高斯分布** | 扩散模型加噪过程、VAE 先验、Latent Diffusion |

## 学习路线

1. 概率空间、随机变量、常见分布
2. 条件概率与贝叶斯公式
3. 期望、方差、协方差
4. 最大似然估计 (MLE) 与最大后验估计 (MAP)
5. 信息论基础（KL 散度、交叉熵）
