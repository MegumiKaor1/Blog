# Adversarial Attacks

> 精心构造微小扰动，让模型输出完全错误的预测。

对抗攻击是 ML 安全最核心的研究方向之一。它揭示了神经网络在高维空间中决策边界的脆弱性——人眼无法察觉的像素级扰动，可以完全改变模型的预测结果。

## 与其他方向的关系

- **Machine Unlearning**：遗忘训练后的模型，对抗鲁棒性如何变化？——开放问题
- **Backdoor Attacks**：后门是「植入的漏洞」，对抗样本是「发现的漏洞」
- **Data Poisoning**：投毒在训练阶段，对抗攻击在推理阶段
- **Model Editing**：编辑模型参数后，是否引入新的对抗脆弱性？
